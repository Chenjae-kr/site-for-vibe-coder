# Site for Vibe Coders

바이브코더에게 유용한 사이트를 카테고리별로 모아두는 웹 큐레이션 저장소입니다.

첫 버전은 [site-for-developers](https://github.com/currenjin/site-for-developers)의 북마크 구조를 참고했고, 초기 데이터는 요청된 [Threads 게시물](https://www.threads.com/@jay_ai__/post/DL34yDsvKIS?xmt=AQF0T2GZuwvsaT8fi_vzK6Nd3Nno-NwawZCvPNscrEfpag)에서 확인한 사이트들로 채웠습니다.

## 현재 포함된 사이트

- Godly
- Awwwards
- CSS Design Awards
- The FWA

## 파일 구조

- `index.html`: 메인 페이지
- `styles.css`: 전체 스타일
- `app.js`: 사이트 데이터와 렌더링 로직

## 실행 방법

정적 파일이라 `index.html`을 브라우저에서 바로 열어도 되고, 간단한 로컬 서버로 띄워도 됩니다.
