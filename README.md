## 🎯 어떤 저장소인가요?

> 바이브코더가 작업 시작 전에 빠르게 참고할 수 있는 대표 사이트를 카테고리별로 모아두는 북마크 저장소입니다.  
> 검색 시간을 줄이고, 좋은 무드와 구현 감각을 동시에 가져올 수 있는 링크를 쌓아갑니다.

현재 구조는 `site-for-developers`의 카테고리형 README 구성을 참고했고, 초기 데이터는 요청하신 Threads 게시물과 추가로 전달해주신 `jay_ai__`의 `2025년 7월 9일` 연속 게시물 사이트 목록을 반영했습니다.

## 🏷️ 라벨 가이드

- 🎯 추천: 가장 먼저 보면 좋은 사이트
- 🧠 영감형: 무드보드, 비주얼, 인터랙션 참고용
- 🧩 실무형: 실제 제품 화면과 패턴 참고용
- 🔄 플로우형: 사용자 여정과 UX 흐름 참고용
- 🎨 크리에이티브형: 실험적 레이아웃, 제너레이티브 아트 참고용
- 🚀 랜딩형: 히어로 섹션, 전환 구조 참고용
- 🧪 인터랙티브형: 실행 가능한 코드와 애니메이션 실험 참고용
- 📊 데이터형: 차트, 시각화 라이브러리 참고용
