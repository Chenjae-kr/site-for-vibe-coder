## 🎯 어떤 저장소인가요?

> 바이브코더가 작업 시작 전에 빠르게 참고할 수 있는 대표적인 사이트를 카테고리별로 모아두는 북마크 저장소입니다.  
> 검색 시간을 줄이고, 좋은 무드와 구현 감각을 동시에 가져올 수 있는 링크를 쌓아갑니다.

현재 첫 버전은 `site-for-developers`의 카테고리형 README 구조를 참고했고, 초기 데이터는 요청된 Threads 게시물에 포함된 UI 레퍼런스 사이트들로 구성했습니다.

## 🏷️ 라벨 가이드

- 🎯 **추천**: 가장 먼저 보면 좋은 사이트
- 🧠 **영감형**: 무드보드, 비주얼, 인터랙션 참고용
- 🏆 **어워드형**: 수상작 중심 레퍼런스
- 🇰🇷 **한국어**: <sub>KR</sub>
- 🇺🇸 **영어**: <sub>EN</sub>

### ⭐ Star

이 프로젝트가 도움이 되셨다면 **Star ⭐**를 눌러주세요. 더 많은 바이브코더들이 쉽게 발견할 수 있습니다.

### 🚀 Contribute

좋은 사이트를 계속 쌓아가려는 저장소입니다. 새로운 링크나 카테고리 제안은 언제든 환영합니다.

**PR**
1. 이 저장소를 `Fork` 합니다.
2. `README.md` 또는 `app.js`에 사이트를 추가합니다.
3. Pull Request를 생성합니다.

**Issue**
- 사이트 추천
- 잘못된 링크 제보
- 카테고리 구조 제안

## 링크

- <span id="recommend">추천</span>
  - [Godly <sub>EN</sub>](https://godly.website/) - 바이브코더가 가장 빠르게 무드보드를 잡을 수 있는 고감도 웹 쇼케이스
  - [Awwwards <sub>EN</sub>](https://www.awwwards.com/websites/ui-design/) - 전 세계 웹 디자인 수상작을 보며 최신 인터랙션과 비주얼 흐름을 확인하기 좋은 대표 아카이브
  - [CSS Design Awards <sub>EN</sub>](https://www.cssdesignawards.com/) - 디자인과 구현 감각을 함께 보고 싶을 때 좋은 CSS/JavaScript 중심 어워드 사이트
  - [The FWA <sub>EN</sub>](https://thefwa.com/awards/) - 실험적이고 몰입감 있는 브랜드 경험 사례를 폭넓게 볼 수 있는 장기 아카이브

- <span id="ui-inspiration">UI 영감 & 레퍼런스</span>
  - [Godly <sub>EN</sub>](https://godly.website/) - 클린한 인터페이스로 좋은 웹 디자인 사례를 빠르게 탐색할 수 있는 쇼케이스
  - [Awwwards <sub>EN</sub>](https://www.awwwards.com/websites/ui-design/) - 혁신적인 UI, 애니메이션, 인터랙션이 반영된 수상작을 지속적으로 볼 수 있는 레퍼런스 허브
  - [CSS Design Awards <sub>EN</sub>](https://www.cssdesignawards.com/) - 실제 프론트엔드 구현 감각까지 참고하기 좋은 웹 디자인 어워드
  - [The FWA <sub>EN</sub>](https://thefwa.com/awards/) - 브랜딩, 실험적 비주얼, 몰입형 웹 경험 레퍼런스를 찾을 때 유용한 아카이브
